# ลงข้อมูลที่ Database

**ข้อมูลอยู่ในโฟลเดอร์งานนะ

1. เปิด XAMPP ----> start Apache + MySQL
2. กด Admin ของ MySQL
3. สร้างตารางชื่อตามไฟล์ database ชนิด utf8_general_ci
4. Import ไฟล์ข้อมูลเรามา
